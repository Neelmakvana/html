Q17:What are the different tags to separate sections of text?

Following tags are used to separate sections of text:

1.<br>→ It separate the line of text by breaking the existing line and taking the text to a new line.
Example:
What are you doing.<br>lets go out.

Output:

What are you doing.
lets go out.

2.<hr>→ It is used to stretch a horizontal rule that is used to separate the content written in two separate lines.
Example:

What are you doing.<hr>lets go out.

Output:

What are you doing.

lets go out.

3.<p>→Paragraph tag is used to write different paragraphs of text on a webpage using HTML.
Example:

<p>What are you doing.<p><p>lets go out.</p>

Output:

What are you doing.

lets go out.

Q19:What is difference between HTML and XHTML?

-HTML stands for Hyper Text Markup Language, and XHTML stand for Extensible Hypertext Markup Language which is a connection between HTML and XML.
-HTML is not case-sensitve so tags and attributes could be written in lower or upper case, And XHTML is case-sensitive and strict with every tags and attributes written in the code should be lower case only.
-HTML files can be stored with .htm,.html extensions; whereas XHTML can be stored with .xhtml,.xht or .xml extensions.

Q20:What are logical and physical tags in HTML?

1. Logical tags:

Logical tag are tags that describe the structure and meaning of the content, without specifying how it should be displayed.

Example: Heading tags(<h1> to <h6>) ; Paragraph tag (<p>) ; Table tag(<table>)

2.Physical tag:

Physical tags are tags that describe how the content should be displayed on the screen.

Example: Bold and strong tag(<b> and <strong>) ; Underline tag (<u> or <ins>)
